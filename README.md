# AHC Tester

AtCoder Heuristic Contest Tester

## How to use (before test)

1. Download AHC Local Tester
2. Move `Cargo.toml`, `Cargo.lock`, `src/**` to `tools`
3. Modify `seeds.txt` to include 1..300 (If needed)
4. Modify `scripts/*.sh`
5. Commit and Push


## How to test

1. Change `main.cpp`
2. Commit and Push
3. See `Actions` Tab on GitHub.com
